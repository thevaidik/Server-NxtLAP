# 🏎️ NxtLAP Racing Server

The NxtLAP server provides a unified racing calendar for major motorsport series. The goal is to provide a single JSON feed for developers to power up beautiful iOS applications.

## 🏁 Available Racing Series

The server automatically tracks and updates scheduling data for the following **9 racing series**:

- **Formula 1** (`formula1`) 
- **IndyCar Series** (`indycar`)
- **MotoGP** (`motogp`)
- **WeatherTech SportsCar Championship** (`imsa`)
- **Super GT** (`supergt`)
- **British GT Championship** (`britishgt`)
- **British Touring Car Championship** (`btcc`)
- **Supercars Championship** (`v8supercars`)
- **World Rally Championship** (`wrc`)
